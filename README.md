# laravel-searcher
Easily search for eloquent model by Laravel

Requirements
------------
 - PHP >= 7.2
 - Laravel = 6.*

Installation
------------
First, install laravel 6.*:

```
composer suhrr/laravel-searcher
```

Then run this command to create search file:

```
php artisan make:search name
```
